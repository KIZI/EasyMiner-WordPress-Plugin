Created Attributes

Attribute (in the sense of this analysis) is a mapping of a domain to finite set of categories. These can be either enumerations, intervals or combination of them. Enumerations usually make sense with categorical domains, which means that the values of the domain cannot be ordered. Example of a categorical domain is sex of person. On the other hand, intervals usually make sense with continuous domains, which means one can tell how far apart two values of the domain are. Example of a continuous domain is age of a person.

This part of the analytical report lists all the created attributes and their characteristics:

  • Column from which the attribute was derived
  • Attribute type (categorical, ordinal, continuous)
  • Frequency analysis – histogram of the intervals or enumerations of the attribute. The histograms are ordered by the highest frequent values. If the attribute contains more categories than 100, the remaining categories are grouped into the “Other” column in the histogram.

The basic building blocks of the task setting is the basic Boolean attributes setting. Basic Boolean attribute setting is a way of construction of basic Boolean attributes (roughly corresponding to items in the apriori algorithm). They are constructed from the set of categories of an attributes. The table below describes the way of construction. Name of the basic Boolean attribute is given by the data mining software and may vary. The column Based on attribute says what the underlying attribute for the basic Boolean attribute is. The column coefficient describes what subsets of the set of categories of the attribute should be taken (the possible values are subsets, intervals, cyclic intervals, cuts, left cuts, right cuts). The last columns state the minimal and maximal length of the basic Boolean attribute.

Derived Boolean attributes are recursive structures built upon the basic Boolean attributes. Conjunction, disjunction and negation of basic Boolean attribute form a derived Boolean attribute. Conjunction, disjunction and negation of derived Boolean attribute form again a derived Boolean attribute.
